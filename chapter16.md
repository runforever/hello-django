## 更多挑战
下面准备了一些新需求，感兴趣的读者可以为自己的博客添加一下功能：

1. Django Admin 添加 Markdown 支持，使用 Markdown 写文章。
2. Django Admin 添加富文本编辑器支持，可以使用百度的富文本编辑器。
3. 文章作者添加 nickname 字段，使用 OneToOneField 扩展 Django User。
4. 博客个人信息介绍支持通过 Django Admin 修改。
